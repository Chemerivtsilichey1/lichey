Умови доступності для навчання осіб з особливими освітніми потребами

Про доступність

Чемеровецький ліцей №1 створює інклюзивне середовище для навчання осіб з особливими освітніми потребами. Ми забезпечуємо рівні можливості для всіх учнів незалежно від їхніх фізичних чи психологічних особливостей.

Наші спеціалісти постійно працюють над вдосконаленням умов для комфортного навчання та соціалізації дітей з особливими потребами.

Інфраструктура

Ліцей обладнаний спеціальними засобами для забезпечення доступності:

  • Пандуси та поручні для маломобільних осіб
  • Спеціальні санвузли
  • Зони релаксації та сенсорного розвантаження

Ми постійно вдосконалюємо нашу інфраструктуру, щоб забезпечити максимальний комфорт для всіх учнів.

Документи

Нормативні документи та матеріали з інклюзивної освіти:

  • Національна стратегія зі створення безбар'єрного простору
  • Заходи щодо впровадження інклюзивної освіти
  • Дорожня карта інклюзії в нашому ліцеї
  • Адаптована освітня програма
